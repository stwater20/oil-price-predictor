# ⛽ 台灣油價預測｜下週中油汽柴油漲跌試算

> 想知道**下週油價會漲還是跌**?這是一個免費的台灣油價預測工具,依台灣中油「浮動油價機制」的 **7D3B 指標**（70% 杜拜原油 ＋ 30% 北海布蘭特，取一週平均），以最新國際原油與匯率，試算下週 **95 / 92 / 98 無鉛與超級柴油**的調價金額，並公開追蹤每一次預測的準確度。

**🔗 線上網站：** https://stwater20.github.io/oil-price-predictor/

---

## ✨ 功能特色

- **下週油價預測**：95 無鉛下週估算牌價與漲跌方向（調漲／調降／持平）。
- **本週油價查詢**：92 / 95 / 98 無鉛、超級柴油的中油參考零售價。
- **歷史油價走勢**：2019 年至今每週零售油價互動圖表，可切換四種油品。
- **近月國際原油**：杜拜、北海布蘭特每日走勢。
- **預測準確度追蹤**：每週記錄預測值，下週以中油官方實際牌價自動比對，累積方向命中率與平均誤差，純客觀、無人工評分。
- **深色／淺色模式**、**社群分享**（LINE／Facebook／X／複製連結）、**RWD 響應式版面**。

## 📊 計算方法

台灣中油採「浮動油價機制」，以 7D3B 指標（70% 杜拜 ＋ 30% 北海布蘭特一週均價）計算週調整幅度。本站以最近一週與前一週的 7D3B 均價差，乘上參考匯率、除以每桶 158.987 公升換算為每公升成本變動，再依汽油浮動反映比例（約 0.9）估算下週牌價。實際牌價另受亞鄰最低價、平穩措施（緩漲）、貨物稅與營業稅等因素影響。

## 🔄 每週自動更新

`.github/workflows/update-data.yml` 每週一自動執行 `fetch-data.mjs`：重抓最新資料、回填上週預測的實際牌價、附加本週最新預測，並提交回 repo，GitHub Pages 隨即更新。也可在 **Actions** 分頁手動觸發（Run workflow）。

## 🗂 資料來源

- 汽柴油零售價、國際原油行情：經濟部能源署「油價資訊管理與分析系統」（oil111）
- 原油報價：路透社
- 台幣兌美元匯率：中央銀行

## 🚀 部署（GitHub Pages ＋ 自訂網域）

1. 將所有檔案推送到 GitHub repo。
2. **Settings → Pages → Source** 選 `Deploy from a branch`，分支 `main`、資料夾 `/ (root)`。
3. 自訂網域：**Settings → Pages → Custom domain** 填入你的網域（GitHub 會自動產生 `CNAME` 檔）。
4. 於 Cloudflare 新增一筆 `CNAME` 記錄指向 `<帳號>.github.io`。首次簽發憑證時建議先設為「DNS only（灰雲）」，待 GitHub Pages 顯示憑證完成後再視需要改回 Proxied，並將 SSL/TLS 模式設為 **Full**。
