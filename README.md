# ⛽ 下週油價預測｜中油汽柴油漲跌 AI 預測

每週六 GitHub Actions 自動抓最新資料、重新訓練一個簡單的線性迴歸模型，預測中油下週一 92／95／98 無鉛汽油與超級柴油的調價金額，並**公開追蹤每一次預測的準確度**。純靜態網站，跑在 GitHub Pages 上。

> ⚠️ 預測僅供參考。實際調價以台灣中油每週日 12:00 公告為準。

## 🧠 模型怎麼運作

中油的浮動油價公式以「7D3B」為指標：**杜拜原油 70%＋布蘭特原油 30%** 的週均價變動，乘上 80% 反映到國內油價。本站不重刻公式，而是讓模型自己從歷史學：

**特徵（3 個）**

1. `7D3B 週均價變動比例 × 前週零售價` — 主要驅動力，對應官方公式的比例調整
2. `max(特徵1, 0)` — 不對稱項，讓模型學到「緩漲」機制（漲的時候打折）
3. `前一次調價金額` — 動能項，捕捉平穩機制的分次補漲

**訓練**：2019 年至今約 400 週的資料，每週重跑一次最小平方法（零相依套件，純 Node 手刻正規方程式）。

**回測表現（誠實揭露）**：汽油 R² ≈ 0.42、平均誤差 ±0.24 元、誤差 ≤0.1 元的比例約 43%。
沒納入匯率與油價平穩機制，凍漲期間會失準——所以網站上有一張預測 vs 實際的對照表，讓數字自己說話。

## 📊 資料來源（全部免費、無需金鑰）

| 資料 | 來源 | 用途 |
| --- | --- | --- |
| 國際原油日價格（杜拜、布蘭特） | [經濟部能源署 油價資訊管理與分析系統](https://data.gov.tw/dataset/6029) | 計算 7D3B 週均價 |
| 中油即時牌價 XML | [台灣中油主產品牌價](https://data.gov.tw/dataset/6339) | 每週回填實際調價、追蹤準確度 |
| 歷史零售油價（2019–今） | 經濟部能源署（一次性建檔於 `data/retail-history.json`） | 訓練資料 |

## 🏗️ 架構

```
index.html                    ← 網站本體（單檔、無外部套件）
data/retail-history.json      ← 每週零售油價歷史（Actions 自動累加）
data/prediction.json          ← 最新預測＋模型指標＋預測紀錄（Actions 產生）
scripts/predict.mjs           ← 抓資料→訓練→預測 一條龍腳本（Node 20+）
.github/workflows/predict.yml ← 週六 23:00 預測、週二 09:00 回填實際（台北時間）
```

## 🚀 部署

1. 推上 GitHub → Actions 頁籤手動跑一次「Predict oil price」
2. Settings → Pages → Deploy from a branch → `main` / root
3. 完成。之後每週六自動預測、週二自動回填實際結果

## ⚙️ 已知限制

- 原油開放資料約有數天延遲，預測使用「最新可得」的 5 個交易日均價
- 未納入台幣匯率（中油公式有匯率項）
- 油價平穩機制（凍漲、緩漲、亞鄰最低價）是人為決策，模型只能從歷史間接學習
- 純學習與趣味用途，不構成任何消費或投資建議
